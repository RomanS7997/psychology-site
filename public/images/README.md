## ⚖️ Юридические требования

### Согласие на публикацию фото/видео детей
Перед публикацией любых фото или видео с детьми необходимо получить письменное согласие родителей. Образец согласия должен включать:
- ФИО родителя и ребенка
- Дата согласия
- Цель использования (для сайта психологического центра)
- Срок действия согласия
- Подпись родителя

### Согласие на публикацию отзывов
Для текстовых и видео-отзывов взрослых клиентов также требуется письменное согласие с указанием:
- ФИО клиента
- Дата согласия
- Разрешение на публикацию отзыва и фото/видео
- Подпись

## 🎨 Оптимизация изображений

Перед загрузкой на сайт рекомендуется оптимизировать изображения:

### Инструменты для оптимизации:
- **TinyPNG** (https://tinypng.com/) - для сжатия PNG и JPG
- **Squoosh** (https://squoosh.app/) - универсальный оптимизатор
- **HandBrake** - для сжатия видео

### Рекомендуемые настройки:
- **Фото:** качество 80-85% для JPG
- **Видео:** битрейт 5-8 Mbps для Full HD
- **Названия файлов:** только латиница, без пробелов (используйте дефисы)

## 🔄 Обновление контента

При обновлении фото/видео:
1. Сохраняйте резервные копии оригиналов
2. Используйте те же имена файлов для замены
3. Очищайте кэш браузера после обновления
4. Проверяйте отображение на мобильных устройствах
